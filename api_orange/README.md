# Project_S9

## Synopsis

An API which simulates the Orange API.

## Motivation

This projet is for one new fonctionality: oppening orange api for all. And so it will be usable for compositon.

## Installation

- Clone this project
- Install the dependencies
- Create a table named "bddOrange"

for run :
- ``
mongod
``
- ``
node app.js
``
